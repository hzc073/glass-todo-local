> 支持局域网访问，多用户，数据不出网

## ✔ 适合谁

- 想要 **私有、可控、不联网也能用** 的 Todo / 日程工具  
- 家庭或小团队，在局域网内部协作  
- 讨厌注册、同步、订阅和分析追踪

---

## 🚀 使用 & 部署

- Windows 本地一键启动  
- 局域网内多设备访问  
- 数据只保存在本地

👉 详细部署说明请查看 `docs/`

---

> 这不是一个“云任务服务”，  
> 它只是一个属于你自己的 Todo。
